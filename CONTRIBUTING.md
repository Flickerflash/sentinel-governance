# Contributing to sentinel-governance

Thank you for your interest in contributing to our governance-as-code project.

## Governance Orientation
All contributions must align with the goal of maintaining high security and project standards. We expect:
- All new checks to be tested.
- Adherence to linting standards.
- Security-first mindset in all code.

## IP Clarity
By contributing to this repository, you agree that your contributions are licensed under the Apache-2.0 license.
